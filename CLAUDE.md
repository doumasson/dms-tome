# DM's Tome

## What Is This
A D&D 5e app for Dungeon Masters. Run campaigns, track combat, roll dice.

## Tech
- React + Vite
- Zustand (state management)

## Features To Build
- Dice roller (all dice types, advantage/disadvantage, modifiers)
- Combat tracker (initiative order, HP tracking, turn management)
- Character sheets (stats, skills, weapons, HP)
- Scene viewer (story text, choices, DM notes)
- Campaign importer (paste JSON, validate, load)
- DM mode toggle (hide/show secrets)

## Design Rules
- Dark theme with gold accents (#ffd700)
- DM-only info has dashed red border
- Big click targets (used at gaming table)
